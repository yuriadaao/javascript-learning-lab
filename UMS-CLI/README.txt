# UMS-CLI

User Management System desenvolvido em Node.js utilizando interface CLI.

## Status

🚧 Em desenvolvimento

## Versão Atual

### v0.2

- Persistência com JSON
- Cadastro de usuários
- Login
- Logout
- Perfil de usuário
- Edição de perfil
- Verificação de perfil
- Exclusão de usuários
- Header dinâmico

## Funcionalidades

- Cadastro de usuários
- Login e Logout
- Persistência de dados em JSON
- Perfil de usuário
- Edição de perfil
- Verificação de integridade do perfil
- Exclusão de usuários
- Sistema de navegação por páginas
- Header dinâmico com informações do usuário
- Controle de sessão através de currentUser

## Estrutura do Usuário

Cada usuário possui:

- id
- fullName
- birthDate
- gender
- adress
- email
- password
- plan
- role
- donations
- fullProfile

## Tecnologias

- JavaScript
- Node.js
- Readline
- File System (fs)
